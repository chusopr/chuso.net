Yes, there are good games for Linux too, although this is the main argument against Linux. Below, a list of games so you can verify it, some of then are quite big and others may seem too simple, but eventhough, being less resource aggressive, are very adictive. These are only a little selection (I may add more in the future, if so, it will be announced in the News column on the right), there are many more you can find in the webpages at the end of this list. Some of them have a version for Windows or MacOS and then you can see their download links, any way, almost all the games have their source code available for download, so you can try to compile it if a version for your platform is not provided. In Gentoo all this games are in Portage or some of its overlays. There also are versions formother systems like BeOS, UN*X, Dreamcast, ... See game web page for more information.

Clásicos
Armagetron advanced
Motorbikes game inspired by Tron movie in which you have to manage other players to crash.
Jump 'n Bump
Super Mario Wars type game
LBreakout
Arkanoid/Breakout clone with a lot of levels and items
Neverball
Parecido a Super Monkey Ball. La idea es: tienes que llegar con la bola al final cogiendo las monedas, pero lo que tu manejas no es la bola, es el escenario. Difícil de entender al principio pero muy adictivo. Incluye juego de golf Neverputt
