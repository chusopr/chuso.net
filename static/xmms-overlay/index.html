Gentoo XMMS overlay

This is an overlay for Gentoo to provide old XMMS 1 ebuilds since they were removed Nov 20, 2006. Actually it is only a backup of old XMMS ebuilds present in Portage until 2006-11-20.

Installation

  1. Go into the directory where you want to store the overlay, for example
    cd /usr/local/overlays
  2. Download the repository
    svn co https://svn.code.sf.net/p/xmms-overlay/code/
  3. Add the directory to PORTDIR_OVERLAY
    grep -q ^PORTDIR_OVERLAY= /etc/make.conf &&
    sed -ie "s,^PORTDIR_OVERLAY=,&\"`pwd`/xmms-overlay \"," /etc/make.conf ||
    echo "PORTDIR_OVERLAY=\"`pwd`/xmms-overlay\"" >> /etc/make.conf

Update

  1. Just go into the overlay directory:
    cd /usr/local/overlays/xmms-overlay
  2. And then execute:
    svn up
